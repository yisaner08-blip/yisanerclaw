# Python 编码风格

## 通用规范

- 缩进：4 空格，禁止 Tab
- 行宽：≤ 100 字符
- 引号：字符串用双引号，单字符用单引号
- 导入：标准库 → 第三方 → 本地，每组间空行

## 类型注解

- 公开函数必须有类型注解
- 使用 `list[dict]` 而非 `List[Dict]`（Python 3.10+）

## 文档字符串

- 模块级：`"""说明 —— 一句话描述"""`
- 函数级：复杂逻辑写说明，简单工具可省略

## 错误处理

- 工具函数返回字符串错误信息，不抛异常
- ⚠️ 避免裸 `except Exception: pass`，至少记录错误
- 外部调用（API、文件、网络）必须 try/except
